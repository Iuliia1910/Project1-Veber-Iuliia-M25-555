# Veber-Iuliia-M25-555
# Лабиринт сокровищ

Лабиринт сокровищ — это текстовая приключенческая игра, 
где игрок исследует комнаты, решает загадки, 
находит предметы и старается открыть сокровищницу.

## Возможности

- Путешествие по лабиринту с комнатами
- Подбор предметов и использование их для прохождения
- Решение загадок для получения наград или открытия сундука
- Случайные события и ловушки
- Условие победы: открыть сокровищницу с помощью ключа или кода

##Игровой процесс

Команды в игре:
go <direction> — перемещение (north/south/east/west)
look — осмотреть текущую комнату
take <item> — подобрать предмет
use <item> — использовать предмет из инвентаря
inventory — показать инвентарь
solve — решить загадку в комнате
quit — выйти из игры
help — показать все команды
